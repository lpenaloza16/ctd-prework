# API Prework Project

This project demonstrates how to connect to and display data from API endpoints.

## Purpose

The primary goal is to show basic functionality for:

- Fetching data from an external API.
- Using a loading state while waiting for a response.
- Rendering the results into the DOM.

## Structure

- **index.html**: Home page for basic environment testing.
- **fakeapi.html**: Integration page containing the `#data-container` for API output.
- **style.css**: Shared layout and navigation styles.
- **script.js**: Logic for API requests and data handling.

## Usage

1. Open `index.html` in a browser.
2. Navigate to the **Apis** link.
3. Observe the transition from "Loading data..." to the fetched API content.
